# Método MP · Mechas Premium — Página de Ventas

Página de vendas (single-page, HTML/CSS/JS puro, sem build) para o **Método MP · Mechas Premium** — curso para coloristas se tornarem referência do mercado. Pós-lançamento, público quente (es-ES).

## Como rodar
Arquivo único: **`index.html`**. É só abrir no navegador ou servir estático:

```bash
python3 -m http.server 8000
# http://localhost:8000/index.html
```

## Stack
- HTML + CSS + JavaScript vanilla, sem dependências de build.
- Tipografia: Fraunces · Lora · Open Sauce Sans (Google Fonts / CDN).
- Animações: reveal on scroll, ondas SVG animadas entre seções, gráficos com fluxo dourado contínuo, CTA glass, count-up, sticky bar.
- Checkout: Hotmart.

## Deploy
Estático — basta servir `index.html` na raiz (Vercel, Netlify, GitHub Pages, etc.).
